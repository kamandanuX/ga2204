# GACS-Ubuntu-22.04
This is autoinstall GenieACS For ubuntu version 22.04 (Jammy)

# Usage
```
sudo su
```
```
git clone https://github.com/safrinnetwork/GACS-Ubuntu-22.04
```
```
cd GACS-Ubuntu-22.04
```
```
chmod +x gacs2204.sh
```
```
sudo apt-get install dos2unix
```
```
dos2unix gacs2204.sh
```
```
bash gacs2204.sh
```
